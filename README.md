# El Tornillo Feliz - Frontend

## 📋 Descripción
Interfaz de usuario para el sistema de gestión de inventario "El Tornillo Feliz". Permite administrar productos (crear, leer, actualizar y eliminar).

## 🚀 Tecnologías utilizadas
- HTML5
- CSS3
- JavaScript (ES6+)
- Fetch API para comunicación con backend

## 📦 Instalación y configuración

### Prerrequisitos
- Navegador web moderno (Chrome, Firefox, Edge)
- Backend del proyecto ejecutándose (ver README del backend)

### Pasos para ejecutar

1. **Clonar el repositorio**
```bash
git clone [https://github.com/mipa57/frontend-tornillo-feliz.git]
cd sistema_inventario_ferreteria/fronted-tornillo-feliz
